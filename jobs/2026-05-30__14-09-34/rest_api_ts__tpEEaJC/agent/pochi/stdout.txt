# Encore REST API with PostgreSQL ## Background Encore is a backend framework for building type-safe distributed systems with declarative infrastructure. In this task, you will build a REST API using Encore.ts with a PostgreSQL database and deploy it to Encore Cloud. ## Requirements * Create a new Encore.ts app named `helloworld-${run-id}` where `run-id` is read from the `ZEALT_RUN_ID` environment variable. * Implement a REST API with CRUD operations for "books" (fields: `id`, `title`, `author`). * POST `/book`: Add a new book. * GET `/book/:id`: Get a book by ID. * GET `/book`: List all books. * Store data using Encore's built-in `SQLDatabase` (PostgreSQL). * Deploy the app to Encore Cloud.
